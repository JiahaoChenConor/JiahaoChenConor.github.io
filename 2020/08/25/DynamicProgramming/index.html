1. 0 - 1 Knapsack Problem

Problem

There are N items and a backpack with a capacity of V.The volume of the i-th item is vi and the value is wi. Find out which items to pack into the backpack, so that the total volume of these items does not exceed the backpack capacity and the total value is the largest. Output the maximum value. Each item can only be used once(0 - 1).

Input :

  1. The two integers in the first line, N and V, are separated by spaces, indicating the number of items and the capacity of the backpack respectively.

  2. Next, there are N lines with two integers vi and wi in each line, separated by spaces, indicating the volume and value of the ith item.

Example:

Thinkings

  1. Define a two-dimensional array f [i] [j].

    • A Set of all choices ：Take the items whose total volume does not exceed j from the first to the i-th item (including the i-th item)

    • f [i] [j] = Maximum value from these choices.

  2. How to get f [i] [j] ?

    1. Split set and find the maximum value on both sides and take the larger one.

  1. For the left part :

    • In order to ensure that no possibility is missed, select items from the first to the i-1th.
    • According to the definition of array, obviously the maximum value is f [i - 1] [j]

  2. For the right part (two cases) :

    • Case 1

      If i-th item is extremely heavy which exceeds the capacity ( wi > capacity means an empty set on the right ), the final maximum value f [i] [j] = f [i - 1] [j] (left maximum)

    • Case 2

      Else, maximum of the right part = f [i - 1] [j - wi] + vi

      Since the i-th item must be included, we add the valuei, at the same time, the weight limit should be reduced weighti for choosing maximum from the rest of items.
