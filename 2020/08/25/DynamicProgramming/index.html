1. 0 - 1 Knapsack Problem

Problem

There are N items and a backpack with a capacity of V.Find out which items to pack into the backpack, so that the total volume of these items does not exceed the backpack capacity and the total value is the largest. Each item can only be used once(0/1).

Input :

  1. The two integers in the first line, N and V, are separated by spaces, indicating the number of items and the capacity of the backpack respectively.

  2. Next, there are N lines with two integers volumni and valuei in each line, separated by spaces, indicating the volume and value of the ith item.

Example:

Thinkings

  1. Define a two-dimensional array dp [i] [j].

    • A Set of all choices ：Take the items whose total volume does not exceed j from the first to the i-th item (including the i-th item)

    • dp [i] [j] = Maximum value from these choices.

  2. How to get dp [i] [j] ?

    1. Split set and find the maximum value on both sides and take the larger one.

  1. For the left part :

    • In order to ensure that no possibility is missed, select items from the first to the i-1th.

    • According to the definition of array, obviously the maximum value is dp [i - 1] [j]

      which means taking maximum value from 1 to i-1 th items and the total vloumn does not

      exceed j.

  2. For the right part (two cases) :

    • Case 1

      If i-th item is extremely big which exceeds the capacity ( volumni > capacity means an empty set on the right ), the final maximum value dp [i] [j] = dp [i - 1] [j] (left maximum)

    • Case 2

      Else, maximum of the right part = dp [i - 1] [j - volumei] + valuei

      Since the i-th item must be included, we add the valuei

      At the same time, the weight limit should be reduced volumni for choosing maximum from the (i-1) items.

Optimization 2 : one-dimensional array

Thinkings

  1. For the optimization 1, we need two lines to update. Can we only use one line to store the answer ?

    • The formula for the question ：dp[i] [j] = max(dp[i - 1] [j], dp[i - 1] [j - volumn[i]] + value[i])

    • In one line is ：dp[j] = max(dp[j], dp[j - volumn[i]] + value[i]); // Delete a dimension means just one line.

If we update from front to back, although dp[j] has no effect, the element dp[j - volumn[i]] we need has been overwritten which has become the element in row i(red), however, what we need is the element in row i - 1(blue).

For example, when updating the third one, dp[j - volumnp[i]] has become red.However we need blue.

So we update from back to front !
